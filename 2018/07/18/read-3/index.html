Recommendation with Social Dimensions

Motivation

  • Most existing social recommender systems treat social relations homogeneously and make use of direct connections
  • weak dependency connections are established among these users when they are not directly connected

Contribution

  • investigate how to exploit the heterogeneity of social relations and weak dependency connections for recommendation
  • employ social dimensions to simultaneously capture heterogeneity of social relations and weak dependency connections, and provide principled ways to model social dimensions
  • propose a recommendation framework SoDimRec which incorporates heterogeneity of social relations and weak dependency connections based on social dimensions

Method

  • Modeling Heterogeneity：One of the most popular approaches to exploit social relations is to assume that the preference of a user should be close to the average preference of her social network

  • Modeling Weak Dependency Connections：Two users with a weak dependence connection are likely to share similar user interests.

Inner Product Similarity Search using Compositional Codes

Motivation

  • Similarity search is a fundamental research topic in the area of computational geometry and machine learning

Contribution

  • propose a compact code approach to approximate inner product similarity search
  • Our approach is based on a vector approximation algorithm, using the composition of several vectors selected from a small set (source dictionary) as the approximation
  • use the indices of the selected vectors to form a compact code to describe the data vector.
  • inner product between the query vector and the database vector can be efficiently estimated from the query vector and the code of the database vector.

Method

  • first producing a compositional dictionary formed by the M-combinations of C that we call source dictionary
  • then finding the best element from the composite dictionary as the approximation of a data vector.
